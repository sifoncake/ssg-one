# LINE/LIFF Terms Review Checklist

Use this checklist to confirm compliance before making the repository public.

## Scope and Use
- [ ] The app usage fits the intended scope of the LINE Messaging API and LIFF.
- [ ] The app does not misrepresent affiliation with LINE or other third parties.

## User Data and Privacy
- [ ] Data collected from LINE is limited to what is needed.
- [ ] User consent is obtained where required (login/connection flows).
- [ ] Data retention and deletion policy is defined.
- [ ] No prohibited sharing or resale of user data.

## Messaging and Content
- [ ] No spam, misleading messaging, or abusive behavior.
- [ ] Message frequency and content comply with LINE policies.

## Branding and Trademarks
- [ ] LINE/LIFF logos or trademarks are used only if permitted.
- [ ] Any required attribution or brand guidelines are followed.

## SDK and Library Use
- [ ] LIFF SDK usage aligns with the official terms.
- [ ] No unauthorized redistribution of SDK assets.

## Publication
- [ ] README includes a note that LINE/LIFF usage is subject to LINE terms.
- [ ] Third-party license notices are available if required.

## References
- LIFF docs: https://developers.line.biz/en/docs/liff/developing-liff-apps/
- LINE terms: https://terms2.line.me/LINE
